---
title: How it works
---
- This is a smaller 6×6 grid using the digits 1–6.
- Every row and every column holds 1–6 with no repeats.
- Each 2×3 **box** also holds 1–6 once.

---
title: Quick example
---
If a row already shows five digits and only `4` is missing, the empty cell must be `4`. The smaller grid makes a gentle warm-up while keeping the same logic as the full puzzle.
