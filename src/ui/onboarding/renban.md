---
title: How it works
---
- All the usual rules apply: rows, columns, and boxes each hold 1–9.
- A purple **path** holds a run of consecutive digits in any order.
- The digits have no gaps, though they need not be sorted along the line.

---
title: Quick example
---
A three-cell path could be `{3,4,5}` or `{6,7,8}`, placed in any arrangement. A four-cell path needs four digits in a row, so it can never include both `1` and `9`.
