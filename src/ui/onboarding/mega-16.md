---
title: How it works
---
- This is a larger 16×16 grid using the digits 1–16.
- Every row and every column holds all sixteen values once.
- Each 4×4 **box** also holds the full set with no repeats.

---
title: Quick example
---
If a 4×4 box is missing only one value, that value must fill the empty cell, just as in a smaller grid. The puzzle is bigger and slower, so take it one box at a time.
