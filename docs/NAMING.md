# Naming & Brand Clearance — Logikku

**Decision: `Logikku` confirmed.** No alternatives needed.
**Domain to register: `logikku.com`** (~$10/yr). Web-only — no `.app` needed.

## Trademark check

Read-only searches across USPTO TESS (US), EUIPO eSearch Plus (EU), and MyIPO (Malaysia) for "Logikku" and phonetic variants (Logiku, Logiqu, Logiqku, Logico, Logikko, Logikko) in classes **9 (software / downloadable apps)** and **41 (entertainment / online games)**.

| Registry | "Logikku" | Variants | Confidence |
|---|---|---|---|
| USPTO TESS | No live mark surfaced | No class 9/41 hits | Medium — TESS UI is JS-rendered; relied on Google site-search + uspto.report indexing |
| EUIPO eSearch | No record | LOGICO exists (educational toys, likely class 28). Logikko = unrelated French hydrogen SAS (liquidated 2023). | Medium — eSearch UI is JS-driven |
| MyIPO Malaysia | No record | None surfaced | Low-medium — MyIPO poorly indexed |

**Verdict: CAUTION-leaning-CLEAR.** No direct Logikku conflicts exist, but `LOGICO` (educational toy brand, EU/global) is phonetically adjacent. A trademark examiner could cite likelihood-of-confusion if Logikku is filed in classes 9/41 covering educational/puzzle software.

**Action: not blocking.** Logikku is web-only (no App Store), so the immediate trademark exposure is low. If the brand grows enough to attract attention (real traffic, revenue, press), commission a paid clearance opinion from a Malaysian and EU TM agent (~RM 2,000–4,000 combined) before formal filing.

## App store / web collision

| Platform | Exact "Logikku" | Close matches in puzzle/game space |
|---|---|---|
| iOS App Store (US/GB/MY) | None | None — adjacent brands (Logic Remote, Logik Academy Pro) are in audio tooling, not games |

| Web | None | Logik.io / Logik.ai (B2B CPQ), Logikk (UK recruiting), Logika Technologies (vision hardware) — all different sectors |

**Verdict: CLEAR.** No collision risk on either store or the web; SEO ranking will be unimpeded.
